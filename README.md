# Numerical Methods Uni Course

This repository contains all of my homework assignments,
completed during my university course on Numerical Methods.

Everything is stored in Jupiter Notebook format because it is very convenient
to prototype and write out some basic mathematical formulas at the same time.

While most of the text and code is my own, I still owe a debt of gratitude to my teacher for developing the course as a whole.

## Basic numerical procedures

In this section I was learning about most basic numerical procedures,
on top of which other more advanced methods are built.

All assignments are in the [file](basic_numerical_procedures.pdf),
where everyone could find some additional tasks to experiment with.

The themes are as follows:
1. [Lagrange interpolation](lagrange_interpolation.ipynb)
2. [Numerical differentiation](numerical_differentiation.ipynb)
3. [Numerical intergration](numerical_integration.ipynb)
4. [Numerical equation solving](numerical_equations.ipynb)
5. [Numerical solution of initial value problem (Cauchy problem)](initial_value_problem.ipynb)
6. [Numerical solution of boundary value problem (Thomas algorithm)](boundary_value_problem.ipynb)
7. [Numerical solution of problem with initial and boundary conditions (leapfrog algorithm)](initial_and_boundary_value_problem.ipynb)

## Numerical methods in equations of mathematical physics

In this section it was time to apply numerical methods to the problems
that are posed by the applied physics,
such as models of string vibrations, thermal conductivity, diffusion, matter transfer and many more.

In the [file](numerical_methods_in_equations_of_mathematical_physics.pdf) you can find lectures and several additional tasks to experiment with.

The themes are as follows
1. [Numerical solution of acoustic wave equation with singularity condition (leapfrog method)](acoustic_wave_equation_with_singularity_condition.ipynb)
2. [Numerical solution of heat equation (Dufort-Frankel method)](heat_equation.ipynb)
3. [Numerical solution of transfer equation (upwind method)](transfer_equation.ipynb)
